# portfolio

## このフォルダの役割

このフォルダは、ポートフォリオサイトとして Git リポジトリにして公開するための場所です。

自身の制作物、実績、プロフィール、活動内容などをまとめ、外部に公開できる形で管理します。

## Claude の基本方針

- 外部公開される前提で、内容の正確さと見え方に注意する
- ポートフォリオサイト本体に必要なファイルを中心に扱う
- 個人用メモやクライアントの内部資料を混ぜない
- 判断に迷う変更は、作業前にユーザーへ確認する
- 削除、移動、リネームは、必ずユーザー承認後にのみ行う

## 作業してよいこと

- ポートフォリオサイトのファイルを作成・更新する
- プロフィール、実績、制作物紹介の内容を整理する
- サイト公開に必要な設定やドキュメントを管理する
- Git リポジトリとしての運用に必要な情報を整える

## ユーザー承認が必要なこと

- ファイルやフォルダの削除
- ファイルやフォルダの移動
- ファイルやフォルダのリネーム
- 公開内容に大きく影響する文章や構成の変更
- 外部公開、公開設定、デプロイ設定に関わる変更
- Git 履歴やリポジトリ構成に影響する操作

## このフォルダに置いてはいけないもの

- 個人発信の下書きや投稿案
- クライアント案件の内部資料
- 学習メモを主目的とする資料
- 生活設計や個人的な記録
- 実験用ツールや便利ツールの置き場
